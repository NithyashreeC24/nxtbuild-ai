# ⚡ NxtBuild AI

NxtBuild AI is an AI-powered website generator that creates responsive, modern HTML websites from natural language prompts using Google's Gemini AI.

## 🚀 Live Demo

[🚀 Live Demo](https://nxtbuild-ai.vercel.app)

> **Note:** The backend is hosted on Render's free tier. The first request after inactivity may take 30–60 seconds while the server wakes up.

## ✨ Features

- 🤖 Generate complete websites using AI
- 👀 Live HTML preview
- 💾 Save generated projects
- ✏️ Rename existing projects
- 🗑️ Delete saved projects
- 📋 Copy generated HTML code
- 📥 Download generated HTML
- 📱 Responsive website generation
- ⚡ Fast React + Vite frontend

## 🛠️ Tech Stack

### Frontend
- React
- Vite
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### AI
- Google Gemini API (Gemini 2.5 Flash)

### Deployment
- Vercel (Frontend)
- Render (Backend)
